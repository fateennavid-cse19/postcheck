PostCheck Authenticator

    How to check the authenticity of your posts using the following links:

  1. Go to this website for generating a public key and a private key
    • Website for Generating Public and Private Key Pair
  2. Save both the public and the private key pair in your notepad. PLEASE DO NOT SHARE THE PRIVATE KEY
  3. Now post the literal text content of the post you want to write, and go this website to output a SHA-256 output
    • SHA-256 Output Generator
